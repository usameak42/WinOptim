- If value doesn't exist or ≠ 0 → call Save-RollbackEntry (Type="Registry", Target="HKLM:\SOFTWARE\Policies\Microsoft\Windows\DataCollection", ValueName="AllowTelemetry", OriginalData=$currentValue.$valueName, OriginalType="REG_DWORD")

- Emit [SUCCESS] with Write-OptLog (Module="Invoke-TelemetryBlock", Operation="Set-ItemProperty", Target="HKLM:\SOFTWARE\Policies\Microsoft\Windows\DataCollection\AllowTelemetry", Values=@{OldValue=$currentValue.$valueName; NewValue=0}, Result="Success", Message="Telemetry capped at Security level", Level="SUCCESS")
- Increment $successCount

CRITICAL: Use Save-RollbackEntry BEFORE Set-ItemProperty (QUAL-02 requirement).

1. **#region Stage 2: AutoLogger Sessions** (TELM-02)

Create array of AutoLogger session names:
```powershell
$autoLoggerSessions = @(
    'AutoLogger-Diagtrack-Listener',
    'DiagLog',
    'SQMLogger'
)
```

For each session:
- Construct registry path: "HKLM:\SYSTEM\CurrentControlSet\Control\WMI\AutoLogger\{$sessionName}"
- Check current state: Get-ItemProperty -Path $regPath -Name "Start" -ErrorAction SilentlyContinue
- Idempotency: If Start value exists and equals 0 → [SKIP], continue
- Save-RollbackEntry (Type="Registry", Target=$regPath, ValueName="Start", OriginalData=$currentValue.Start, OriginalType="REG_DWORD")
- Set-ItemProperty -Path $regPath -Name "Start" -Value 0 -Type DWord -ErrorAction Stop
- Write-OptLog with SUCCESS, increment $successCount

Handle CONTEXT error case: "AutoLogger Session Failures: Skip failed session, continue disabling other sessions"
```powershell
try {
    Set-ItemProperty ... -ErrorAction Stop
} catch {
    Write-Host "[WARNING] Failed to disable AutoLogger session $sessionName: $_" -ForegroundColor Yellow
    Write-OptLog ... -Result "Warning" -Level "WARNING"
    $warningCount++
    continue
}
```

2. **#region Stage 3: Telemetry Services** (TELM-03)

Load services from config:
```powershell
$config = Get-Content "$PSScriptRoot\..\config\services.json" | ConvertFrom-Json
$telemetryServices = $config.disabled | Where-Object { $_.name -like 'DiagTrack' -or $_.name -like 'dmwappushservice' }
```

For each service:
- Get-Service -Name $serviceName -ErrorAction SilentlyContinue
- If null → [SKIP] "Service not found", continue
- Check current state: Get-WmiObject Win32_Service -Filter "Name='$serviceName'" | Select-Object StartMode, Status
- Idempotency: If StartMode='Disabled' AND Status='Stopped' → [SKIP], continue
- CONTEXT: "Service State Mismatch" - If StartType matches but Status doesn't, log WARNING
- Prompt user: "Disable this service? (Y/N/A for All)" per CONTEXT decision
- Save-RollbackEntry (Type="Service", Target=$serviceName, OriginalStartType=$currentStartType)
- Stop service if running: Stop-Service -Name $serviceName -Force -ErrorAction Stop
  - On error: CONTEXT "Service Stop Timeout" → Prompt user "Force kill service process? (Y/N)"
- Set-Service -Name $serviceName -StartupType Disabled -ErrorAction Stop
- Write-OptLog with SUCCESS, increment $successCount

Handle CONTEXT error case: "Service Disable Failures: Prompt user to continue or halt on each service disable failure"
```powershell
try {
    Set-Service ... -ErrorAction Stop
} catch {
    Write-Host "[ERROR] Failed to disable $serviceName: $_" -ForegroundColor Red
    $continue = Read-Host -Prompt "Continue with remaining services? (Y/N)"
    if ($continue -ne 'Y' -and $continue -ne 'y') {
        break
    }
    $errorCount++
    continue
}
```

3. **#region Stage 4: Scheduled Tasks** (TELM-04)

Create array of telemetry tasks:
```powershell
$tasks = @(
    @{ Name = 'CompatibilityAppraiser'; Path = '\Microsoft\Windows\Application Experience\CompatibilityAppraiser' },
    @{ Name = 'ProgramDataUpdater'; Path = '\Microsoft\Windows\Application Experience\ProgramDataUpdater' },
    @{ Name = 'Consolidator'; Path = '\Microsoft\Windows\Customer Experience Improvement Program\Consolidator' },
    @{ Name = 'UsbCeip'; Path = '\Microsoft\Windows\Customer Experience Improvement Program\UsbCeip' }
)
```

Prompt user for strategy per CONTEXT "Scheduled Task Handling":
```powershell
Write-Host "`n[Scheduled Task Handling Strategy]" -ForegroundColor Cyan
Write-Host "1. Disable only: Sets State=0, fully reversible" -ForegroundColor White
Write-Host "2. Delete task: Removes task, cleaner but irreversible" -ForegroundColor White
Write-Host "3. Hybrid: Disable custom tasks, delete system telemetry tasks" -ForegroundColor White
$strategy = Read-Host -Prompt "Choose strategy (1/2/3)"
```

For each task:
- Get-ScheduledTask -TaskName $taskName -ErrorAction SilentlyContinue
- If null → [SKIP] "Task not found", continue
- CONTEXT: "Task Checks" - Check State + Enabled property
  - If State='Disabled' AND -not $Enabled → [INFO] "Already disabled, confirming..."
- CONTEXT: "System vs Custom" - Distinguish by task author/creator: $taskAuthor = $scheduledTask.Author; $isSystemTask = $taskAuthor -like '*Microsoft*' -or $taskPath -like '*\Microsoft\Windows\*'
- Apply strategy:
  - Strategy 1: Disable-ScheduledTask -TaskName $taskName -TaskPath (Split-Path $taskPath)
    - Save-RollbackEntry (Type="ScheduledTask", Target=$taskPath, ValueName="State", OriginalData=$currentState, OriginalType="TaskState")
  - Strategy 2: Unregister-ScheduledTask -TaskName $taskName -TaskPath (Split-Path $taskPath) -Confirm:$false
    - Save-RollbackEntry (Type="ScheduledTask", Target=$taskPath, ValueName="TaskDefinition", OriginalData=$scheduledTask.TaskDefinition.OuterXml, OriginalType="TaskXml")
  - Strategy 3: Hybrid - If $isSystemTask → Delete; Else → Disable
- Write-OptLog with SUCCESS, increment $successCount

Handle CONTEXT error case: "Scheduled Task Failures: Continue disabling remaining tasks, list failed tasks at end"
```powershell
try {
    Disable-ScheduledTask ... -ErrorAction Stop
} catch {
    Write-Host "[ERROR] Failed to process task '$taskName': $_" -ForegroundColor Red
    Write-OptLog ... -Result "Error" -Level "ERROR"
    $errorCount++
    continue
}

# After loop
if ($errorCount -gt 0) {
    Write-Host "`n[WARNING] Failed to process $errorCount task(s). Review log for details." -ForegroundColor Yellow
}
```
